System message

The purpose of this website is to store one or more editions of work datasets, and check their quality. Potentially, the system can be integrated with automatic uploads from IDM3 tools as their datasets get created as a result of the FME process.

The site is only open to uploaders with an Eionet account.

  • List of datasets
  • Upload new dataset

Limitations of this prototype

  • Only works if you upload MDB or ACCDB files.
  • Some navigational links are missing or don't work.
  • Confusion in terminology.
  • Only tableExists and sqlCheck QA tests work at the moment
